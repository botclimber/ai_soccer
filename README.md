# ai_soccer
AI to predict football matches

Technical Info:
	- Layes & Neurons:
		- 2 Layers with 19 Neurons
	
	- Activation function:
		- relu
	
	- Optimizer:
		- Adagrad

	- Learning rate:	
		- 0.01
	
	- steps
		- 10.000

	- classes:
		- 3
		- ['H', 'D', 'A']

## Dataset

Our dataset its composed by 24 variable inputs.

Install dependencies:
